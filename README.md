# Лабораторная работа по индексации файлов

## Disclaimer

Данный код не является хорошим показателем.
Здесь не уделено большое внимание обработке ошибок.

Цель проекта -- показать ускорение чтение файла при помощи построение индекса и использование системного вызова [lseek](https://man7.org/linux/man-pages/man2/lseek.2.html).

Автор не осилил такую задачу на C, поэтому выполнил её на Rust.

Все данные были получены из открытых источников и на момент создания проекта были доступны в поисковой системе [Google](https://google.com)

## Скачивание репозитория

```shell
$ git clone https://github.com/assisken/os-files-index.git
...
$ cd os-files-index
```

## Построение индекса

```shell
$ ./lab4 --build            
Building index of books.csv to index.csv
Done!
```
